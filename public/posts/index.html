Building an Azure Static Web App

Azure Static Web Apps is a service which is currently in preview on Azure. Because its currently in preview, the service is free until it leaves preview. This basically gives you an website, SSL cert (curtesy of Microsoft) and free web hosting! Azure Static Web Apps automatically builds and deploys a full stack web app from just an github repository. These apps only consist of a static web frontend, and an backend which uses Azure Functions.
